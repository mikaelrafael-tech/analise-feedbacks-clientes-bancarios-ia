# 📋 Metodologia

## Objetivo

O objetivo deste projeto foi aplicar Inteligência Artificial Generativa e Engenharia de Prompt para analisar feedbacks de clientes bancários, transformando dados em informações estratégicas para apoiar a tomada de decisão.

---

# Metodologia Utilizada

O desenvolvimento do projeto foi realizado em sete etapas:

## 1. Organização da Base de Dados

Foi utilizada uma base de dados contendo 100 feedbacks de clientes bancários nos formatos CSV e Excel.

Cada registro continha as seguintes informações:

- ID
- Categoria
- Canal
- Nota
- Feedback

Essa organização permitiu que a Inteligência Artificial interpretasse corretamente os dados.

---

## 2. Definição do Problema

Foi definido como objetivo principal compreender a percepção dos clientes sobre os serviços oferecidos pela instituição financeira.

A análise buscou responder perguntas como:

- Quais serviços apresentam maior satisfação?
- Quais categorias concentram mais reclamações?
- Quais padrões podem ser identificados?
- Quais oportunidades de melhoria podem ser propostas?

---

## 3. Engenharia de Prompt

Foi desenvolvido um prompt estruturado para orientar a Inteligência Artificial a atuar como uma consultoria especializada em:

- Inteligência Artificial
- Business Intelligence (BI)
- Análise de Dados
- Customer Experience (CX)

O prompt definiu regras para garantir que todas as análises fossem realizadas exclusivamente com base nos dados fornecidos.

---

## 4. Processamento da Base

A base de dados foi analisada pela Inteligência Artificial para identificar:

- Indicadores de desempenho (KPIs)
- Padrões de comportamento
- Tendências
- Reclamações recorrentes
- Pontos fortes
- Oportunidades de melhoria

---

## 5. Geração do Relatório Executivo

Com base na análise, foi produzido um relatório contendo:

- Resumo Executivo
- KPIs
- Análise de Sentimentos
- Insights Estratégicos
- Plano de Ação
- Recomendações para a Diretoria

Todas as conclusões foram fundamentadas exclusivamente nas informações presentes na base de dados.

---

## 6. Documentação do Projeto

Todo o desenvolvimento foi organizado em um repositório GitHub contendo:

- Base de dados
- Prompt utilizado
- Relatório Executivo
- Documentação técnica
- README detalhado

Essa estrutura facilita a compreensão e a reprodução do projeto.

---

## 7. Conclusão

A utilização da Inteligência Artificial demonstrou como técnicas de Engenharia de Prompt podem apoiar a análise de dados e a geração de informações estratégicas.

Além da análise dos feedbacks, o projeto permitiu desenvolver competências em documentação técnica, organização de projetos, GitHub e comunicação de resultados.
