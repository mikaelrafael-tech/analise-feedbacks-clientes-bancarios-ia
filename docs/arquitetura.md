# 🏗️ Arquitetura do Projeto

## Visão Geral

Este projeto foi desenvolvido para demonstrar a aplicação de Inteligência Artificial Generativa na análise de feedbacks de clientes bancários.

O fluxo do projeto segue uma sequência organizada, desde a preparação dos dados até a geração de insights estratégicos.

---

# Fluxo do Projeto

```text
Base de Dados (CSV / Excel)
            │
            ▼
 Engenharia de Prompt
            │
            ▼
 Inteligência Artificial
            │
            ▼
 Análise dos Dados
            │
            ▼
 Relatório Executivo
            │
            ▼
 Documentação no GitHub
```

---

# Componentes do Projeto

## Base de Dados

Contém os feedbacks utilizados durante a análise.

Arquivos:

- feedback_clientes.csv
- feedback_clientes.xlsx

---

## Engenharia de Prompt

Responsável por orientar a Inteligência Artificial durante a interpretação dos dados.

Arquivo:

- prompt_executivo.md

---

## Inteligência Artificial

A IA foi utilizada para:

- interpretar os dados;
- calcular indicadores;
- identificar padrões;
- gerar insights;
- elaborar recomendações.

---

## Relatório Executivo

Documento final contendo:

- Resumo Executivo;
- KPIs;
- Análise de Sentimentos;
- Insights Estratégicos;
- Plano de Ação;
- Recomendações.

---

## Documentação

Todo o desenvolvimento foi organizado utilizando GitHub, permitindo rastreabilidade, organização e fácil manutenção do projeto.

---

# Benefícios da Arquitetura

- Organização dos arquivos.
- Facilidade de manutenção.
- Separação entre dados, documentação e resultados.
- Estrutura reutilizável para futuros projetos.
