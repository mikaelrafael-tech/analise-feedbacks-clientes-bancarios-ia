# 📚 Aprendizados do Projeto

## Objetivo

Registrar os principais conhecimentos adquiridos durante o desenvolvimento deste projeto.

---

# Conhecimentos Técnicos

Durante o projeto foram praticados os seguintes conceitos:

- Inteligência Artificial Generativa
- Engenharia de Prompt
- Business Intelligence (BI)
- Análise de Dados
- Customer Experience (CX)
- Documentação Técnica
- Git
- GitHub
- Markdown

---

# Principais Aprendizados

## Organização dos Dados

Aprendi a estruturar uma base de dados para facilitar sua interpretação pela Inteligência Artificial.

---

## Engenharia de Prompt

Compreendi a importância de criar prompts claros, estruturados e objetivos para obter respostas mais precisas.

---

## Análise de Dados

Aprendi a interpretar indicadores, identificar padrões e transformar informações em recomendações estratégicas.

---

## Documentação

Percebi que documentar o projeto é tão importante quanto realizar a análise, pois facilita o entendimento e a reutilização do trabalho.

---

## GitHub

Aprendi a organizar um repositório profissional utilizando pastas, arquivos Markdown e boas práticas de documentação.

---

Este projeto contribuiu para o desenvolvimento das minhas habilidades em Inteligência Artificial, Análise de Dados e documentação técnica, reforçando a importância da organização e da interpretação dos dados para apoiar a tomada de decisão.
